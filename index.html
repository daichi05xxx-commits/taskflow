未設定
▸ Supabaseの設定手順を見る（初回のみ）
① supabase.com で無料アカウントを作成し、新しいプロジェクトを作ります。
② プロジェクトの Settings > API から「Project URL」と「anon public key」をコピーして上に貼り付けます。
③ Table Editor の SQL Editor で以下のSQLを実行します： 
create table user_data (
  user_id uuid primary key references auth.users on delete cascade,
  tasks jsonb default '[]',
  todos jsonb default '[]',
  schedules jsonb default '[]',
  settings_data jsonb default '{}'
);
alter table user_data enable row level security;
create policy "own" on user_data for all
  using (auth.uid() = user_id)
  with check (auth.uid() = user_id);

④ 上のフォームにURLとキーを入力 → ログインまたは新規アカウント作成。
⑤ 他のデバイスでも同じURL・キーを入力して同じメールアドレスでログインするとデータが同期されます。
